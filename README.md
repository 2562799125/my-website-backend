# 矿山可视化监管平台 v1.0.0

## 简介
矿山生态修复智能监测与碳汇评估平台是一个基于Web的可视化监管系统，用于监测矿山生态修复进展和碳汇评估。

## 系统要求
- macOS 10.15 或更高版本
- Python 3.7 或更高版本

## 安装和运行

### 方法一：使用macOS应用包（推荐）
1. 双击 `矿山可视化监管平台.app` 启动应用
2. 系统会自动启动Web服务器并打开浏览器

### 方法二：使用命令行脚本
1. 双击 `矿山可视化监管平台.command` 启动应用
2. 或在终端中运行：`python3 app_launcher.py`

### 方法三：手动启动
1. 在终端中进入应用目录
2. 运行：`python3 app_launcher.py`
3. 在浏览器中访问显示的地址（通常是 http://localhost:8000）

## 功能特性
- 🗺️ 高德地图集成，支持2D/3D地图切换
- 📊 实时数据监控和图表展示
- 🏔️ 多矿山位置管理和快速跳转
- 📈 碳汇评估和生态修复进度追踪
- 🎨 现代化用户界面设计
